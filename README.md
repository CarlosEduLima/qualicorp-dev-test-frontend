# Qualicorp Dev Test
> Frontend CRUD de usuários 

## Instalação local

Windows:

* Clone o projeto 
* No diretório raiz rode os sequintes comandos:

```sh
npm install
npm run serve
```

## Tecnologias

* Vue.js
* Vuex
* vue-toast-notification
* vue-router
* axios
